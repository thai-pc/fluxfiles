# Đề xuất: Detail panel cho thư mục + cải thiện UX/UI

## 1. Khi chọn thư mục — 4 nút nên giữ nút nào?

### Hiện tại (4 nút cho cả file và folder)
| Nút | File | Folder |
|-----|------|--------|
| **Select** | Chọn file, gửi FM_SELECT | Có thể dùng khi embed (chọn path) |
| **Copy URL** | Copy URL file | Folder không có URL → vô nghĩa |
| **Download** | Tải file | Folder không download được → vô nghĩa |
| **Delete** | Xóa file | Xóa folder → có nghĩa |

### Đề xuất: Chỉ hiển thị nút có ý nghĩa

**Với thư mục (folder):**
- ✅ **Select** — Giữ (khi embed, parent có thể dùng path để navigate)
- ❌ **Copy URL** — Ẩn (folder không có URL)
- ❌ **Download** — Ẩn (folder không download)
- ✅ **Delete** — Giữ (có thể xóa folder)

**Với file:**
- Giữ cả 4 nút như hiện tại

**Cách implement:** Thêm `x-show="detailFile?.type !== 'dir'"` cho Copy URL và Download. Select đổi label thành "Select" cho file, "Select folder" cho folder (hoặc dùng chung "Select").

## 2. Tab SEO "kỳ kỳ" — vấn đề và giải pháp

### Vấn đề
- Tab **SEO** hiện hiển thị cho **mọi loại file** (PDF, video, folder…)
- SEO metadata (title, alt text, caption) chỉ có ý nghĩa với **ảnh**
- Với folder/PDF/video → tab SEO trống hoặc không phù hợp

### Đề xuất

**Option A (khuyến nghị): Chỉ hiện SEO cho ảnh**
- Giống AI và Crop: `x-show="detailFile && isPreviewable(detailFile, 'image')"`
- Tab SEO chỉ xuất hiện khi chọn file ảnh
- Giao diện rõ ràng, không gây nhầm lẫn

**Option B: Đổi tên + ẩn khi không phải ảnh**
- Đổi "SEO" → "Metadata" hoặc "Image metadata"
- Vẫn chỉ hiện với ảnh
- Giúp user hiểu đây là metadata cho ảnh

**Option C: Giữ cho mọi file, thêm fallback**
- Với file không phải ảnh: hiện message "Metadata only available for images"
- Tốn không gian, ít giá trị → không nên dùng

## 3. Cải thiện UX/UI khác

### 3.1 Info tab khi chọn folder
- **Size** — Folder thường không có size rõ ràng → hiện "—" hoặc ẩn dòng
- **Modified** — Giữ
- **Disk** — Giữ
- **Path** — Giữ

### 3.2 Preview thumbnail khi chọn folder
- Đã dùng `thumbIcon(detailFile)` → hiện 📁
- Có thể tăng kích thước icon folder để dễ nhận biết

### 3.3 Tags / AI tags khi chọn folder
- **detail-tags** (badge + AI tags): AI tags chỉ cho ảnh → ẩn phần AI tags khi folder
- Badge disk vẫn hiển thị

### 3.4 Select button label
- File: "Select File"
- Folder: "Select folder" hoặc "Select" (nếu context đã rõ)
- Cần thêm key i18n: `common.select_folder`

## 4. Tóm tắt thay đổi đề xuất

| Thay đổi | Mô tả |
|----------|-------|
| Ẩn Copy URL khi folder | `x-show="detailFile?.type !== 'dir'"` |
| Ẩn Download khi folder | `x-show="detailFile?.type !== 'dir'"` |
| Chỉ hiện tab SEO cho ảnh | `x-show="detailFile && isPreviewable(detailFile, 'image')"` |
| (Tùy chọn) Đổi "SEO" → "Metadata" | Cập nhật lang |
| Ẩn AI tags khi folder | Đã ẩn vì tags chỉ cho ảnh |
| Size = "—" khi folder | `formatSize` trả về "—" nếu type dir |
| Label Select theo loại | "Select File" / "Select folder" |
